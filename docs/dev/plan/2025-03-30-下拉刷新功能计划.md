# 下拉刷新功能计划

> 本文档记录 client 项目列表页面下拉刷新功能的实现计划。

## 概述

**创建时间**: 2025-03-30
**优先级**: 中
**影响范围**: frontend/client 列表页面

---

## 需求描述

在 frontend/client 项目的列表展示数据中，添加列表下拉刷新功能：
- 只有列表已经滚动到顶部时，才可以触发下拉刷新
- 刷新时有视觉反馈（下拉指示器和加载动画）
- 支持触摸和鼠标操作（适配桌面端和移动端）

---

## 技术方案分析

### 候选方案

| 方案 | 优点 | 缺点 |
|------|------|------|
| vant PullRefresh | 功能完善、交互体验好、成熟稳定 | 需要引入 vant 库 |
| vue3-pull-to-refresh | 专门的下拉刷新库 | 社区较小、可能存在兼容性问题 |
| 自定义组件实现 | 风格一致、轻量、可控 | 需要自行实现，复杂度高 |

### 选定方案

使用 **vant PullRefresh** 组件：

- vant 是成熟的 Vue 3 移动端 UI 库，PullRefresh 组件功能完善
- 支持触摸和鼠标操作，自动处理 scrollTop 检测
- 可通过 CSS 自定义样式以适配项目 glass 风格
- 按需引入可减少包体积

---

## 关键问题处理

### 问题一：双层滚动容器

当前项目存在双层滚动结构：
```
MainLayout.vue (line 40, overflow-y-auto)  ← 外层
  └─ MomentsView.vue (line 354, scrollContainer overflow-y-auto)  ← 内层
```

**解决方案**：下拉刷新组件放置在**内层滚动容器**上
- vant PullRefresh 通过检测内部滚动容器的 scrollTop 来判断是否可触发
- 内层容器 scrollTop === 0 时才能下拉刷新，满足需求

### 问题二：子组件刷新事件传递

AlbumsView 使用 AlbumList/PhotoList 子组件展示列表：
- 子组件触发刷新 → emit('refresh') → 父组件重新加载数据
- 需要在子组件中添加 `@refresh` 事件监听

### 步骤一：安装依赖

**文件**: `frontend/client/package.json`

```bash
cd frontend/client && pnpm add vant
```

### 步骤二：创建样式适配文件

**文件**: `frontend/client/src/styles/vant-pull-refresh.css`

**改动说明**:
- 覆盖 vant PullRefresh 默认样式
- 使用项目 glass 风格
- 自定义下拉指示器外观

### 步骤三：应用到列表页面

**目标页面**（排除非列表页面）：

| 页面 | 文件路径 | 滚动容器 | 父组件 |
|------|----------|----------|--------|
| 动态列表 | `views/MomentsView.vue` | line 354 | 无 |
| 相册列表 | `views/components/AlbumList.vue` | line 4 | AlbumsView |
| 照片列表 | `views/components/PhotoList.vue` | line 19 | AlbumsView |
| 纪念日列表 | `views/AnniversariesView.vue` | line 373 | 无 |
| 地点列表 | `views/PlacesView.vue` | line 328 | 无 |
| 通知列表 | `views/NotificationsView.vue` | line 78 | 无 |

**改动说明**:
- 在滚动容器外层包裹 `<van-pull-refresh>` 组件
- 添加 `v-model="loading"` 控制加载状态
- 添加 `@refresh` 事件处理

### 步骤四：处理数据刷新逻辑

**文件**: 各列表页面

**改动说明**:
- 添加 `isRefreshing` ref 状态
- 刷新时重置页码为 1，重新加载数据
- 子组件通过 emit 向父组件传递刷新事件

**示例代码**:
```vue
<template>
  <van-pull-refresh v-model="isRefreshing" @refresh="handleRefresh">
    <div ref="scrollContainer" class="overflow-y-auto ...">
      <!-- 列表内容 -->
    </div>
  </van-pull-refresh>
</template>

<script setup>
const isRefreshing = ref(false);

const handleRefresh = async () => {
  currentPage.value = 1;
  await fetchMoments(1);
  isRefreshing.value = false;
};
</script>
```
